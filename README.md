# ue4-deepdrive-deps
Prebuild Deepdrive + Unreal Editor in docker and export it to your Linux system

## Prerequisites
- NVIDIA Docker 2
- [ue4-full image](https://adamrehn.com/docs/ue4-docker/use-cases/linux-installed-builds)

```
pip install -r requirements.txt
```

## Build

```
python package.py
```
